  • 商標・登録商標について

本ヘルプに記載されている各社の会社名、製品名に関する登録商標および商標は以下のとおりです。

  • Android™、Google Play™、Chrome™、Google Chrome™は、Google Inc. の商標です。
  • Apache®、Ant、Cordova、Apache Ant™、Apache Cordova™は 、アメリカ合衆国および/またはその他の国における Apache Software Foundationの商標または登録商標です。
  • App Store℠、iPhone®、iPad®、iTunes®、Keychain®、Mac®、Safari®、Xcode®は、米国および他の国々で登録されたApple Incの商標です。
  • Intelは、アメリカ合衆国および/またはその他の国における Intel Corporation の商標です。
  • iOSは、米国およびその他の国におけるCisco社の商標または登録商標です。
  • Microsoft®、Windows®、Internet Explorer®は、米国 Microsoft Corporationの米国、日本およびその他の国における登録商標または商標です。
  • Node.jsは、Joyent, Inc.の商標であり、使用許諾を得て使用しています。
  • Oracle、Javaは、Oracleおよび/またはその関連会社の登録商標です。
  • PythonはPython Software Foundationの登録商標です。
  • その他記載されている会社名、製品名は、各社の商標または登録商標です。
